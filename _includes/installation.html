Installation

Cask

It is highly recommended to install Ecukes is by using Cask. Install Cask then add a Cask file looking something like this. 

(source melpa)

(package "super-duper" "0.0.1" "Super Duper.")

(development
  (depends-on "ecukes")
  (depends-on "espuds"))

Next, run the cask command to install all dependencies.

A simple way to run your features is to create a Makefile. 

all:
	cask exec ecukes features
